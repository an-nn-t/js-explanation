Node.jsとは

Node.jsは、ChromeのV8エンジンを使用してJavaScriptをサーバーサイドで実行する環境です。

基本的な使い方

npm init -y

// 簡単なHTTPサーバー

非同期処理（Promise/async-await）

非同期処理の基本であるPromiseと、より読みやすいasync/await構文を実用的な例で解説します。

中級 14分

Promiseの基本

// Promise の作成
const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve("データを取得しました");
    }, 1000);
  });
};

// Promise の使用
fetchData()
  .then(data => console.log(data))
  .catch(error => console.error(error));

async/await

// async/await を使用した書き方
async function getData() {
  try {
    const data = await fetchData();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}

// 複数の非同期処理
async function multipleRequests() {
  const [user, posts] = await Promise.all([
    fetchUser(),
    fetchPosts()
  ]);

  return { user, posts };
}

パフォーマンス最適化

不要なループ処理の削減、関数のメモ化、描画の最適化など、実用的な高速化テクニックを紹介します。

上級 17分

メモ化の例

function memoize(func) {
  const results = {};
  return (...args) => {
    const argsKey = JSON.stringify(args);
    if (!results[argsKey]) {
      results[argsKey] = func(...args);
    }
    return results[argsKey];
  };
}

const expensiveFunction = (num) => {
  console.log('Calculating...');
  return num * 2;
};

const memoizedExpensiveFunction = memoize(expensiveFunction);
console.log(memoizedExpensiveFunction(5)); // Calculates
console.log(memoizedExpensiveFunction(5)); // Uses cached result
